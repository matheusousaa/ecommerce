## üß∞ Tech Stack

**Backend:**
- Laravel 10
- PostgreSQL (or MySQL-compatible)
- Eloquent ORM

**Frontend:**
- Vue 3
- Inertia.js
- Tailwind CSS
- Vite

**Other:**
- Authentication via Laravel Breeze (Inertia stack)
- File upload via Laravel Storage
- Session & database cache
- `.env` based environment configuration
